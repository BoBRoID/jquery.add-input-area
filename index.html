jquery.addInputArea.3.1.js

Last update:2012-07-09

You can add or delete the list of text-box or text-area.

テキストボックス、またはテキストエリアを動的に追加・削除します。

Basic 基本 


(HTML)
<ol id="list1">
	<li class="list1_var">
		<input type="text" size="40">
		<button class="list1_del">Delete</button>
	</li>
</ol>
<input type="button" value="Add" class="list1_add">

(JavaScript)
$('#list1').addInputArea();

You can save time and effort by following naming convention.

増減させる対象のエリア、追加ボタン、削除ボタンのCSSクラス名は、
命名規則に従うことでプラグインに渡すオプションの数を減らすことができます。
詳しくは'README'をご覧ください。

Options オプション 


(HTML)
<ol id="list2">
	<li class="var_area02">
		<textarea
			name_format="data[posts][mail][%d]"
			name="data[posts][mail][0]">
		</textarea>
		<button class="del_button02">Delete</button>
	</li>
</ol>
<input type="button" value="Add!" class="add_button02">

(JavaScript)
$('#list2').addInputArea({
	area_var : '.var_area02',
	btn_add  : '.add_button02',
	btn_del  : '.del_button02'
});

You can set options for plug-in.
Please look at "README" for details.

プラグインにはオプションを渡すことができます。
詳しくは README をご覧ください。

The ornament by CSS and images CSSや画像による装飾

Add
(HTML)
<ul id="list3">
	<li class="list3_var">
		<input type="text" size="40" name="list3_0">
		<img class="list3_del" src="img_del.png">
	</li>
</ul>
<a href="javascript:void(0)" class="list3_add">Add</a><br>

(CSS)
.list3_add {
	display:block;
	width:100px;
	padding:4px 8px;
	text-decoration:none;
	text-align:center;
	color:#fff;
	background:#24b;
	border:2px solid #37f;
	border-radius:4px;
}

(JavaScript)
$('#list3').addInputArea();

You can decorate add-button and delete-button with CSS or images.

追加ボタン、削除ボタンはCSSや画像で装飾できます。

The list which contains some input areas 複数の入力欄を包含する増減エリア 


(HTML)
<table id="list4">
	<tbody>
		<tr class="list4_var">
			<td>
				<label for="list4_name_0">Name</label>
				<input type="text" size="40" name="list4_name_0">
			</td>
			<td>
				<label for="list4_mail_0">Mail</label>
				<input type="text" size="40" name="list4_mail_0">
			</td>
			<td class="del_area04">
				<button class="list4_del">Delete</button>
			</td>
		</tr>
	</tbody>
</table>
<input type="button" value="Add" class="list4_add"><br>

(JavaScript)
$('#list4').addInputArea({
	area_del : '.del_area04'
});

The list can be fluctuated even if how many input-area were contained.

複数の入力欄を扱う場合も、特にオプションを必要としません。
