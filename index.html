jquery.addInputArea.2.0.js

最終更新日:2012-05-10

テキストボックス、またはテキストエリアを動的に追加・削除します。
ver2.0から、複数の入力欄を一括して増減できるようになりました。

基本


(HTML)
<ol id="list1">
	<li><input type="text" size="40"></li>
</ol>
<input type="button" value="項目を追加" id="list1_add">

(JavaScript)
$('#list1').addInputArea();

設定が必要な値は、リストのid属性と、追加ボタンのid属性です。
追加ボタンのid属性は、" (リストのid属性)+'_add' "の形式で記述することで、
プラグインに渡すオプションの数を減らすことができます。

オプションを指定


(HTML)
<ol id="list2">
	<li><textarea>あにゃまる探偵キルミンずぅ DVD 第1巻</textarea></li>
</ol>
<input type="button" value="Add!" id="add_button02">

(JavaScript)
$('#list2').addInputArea({
	btn_add : '#add_button02',
	btn_del : $('<button>').text('Delete!'),
});

プラグインにはオプションを渡すことができます。
詳しくは README をご覧ください。

CSS、画像による装飾

項目を追加
(HTML)
<ul id="list3">
	<li><input type="text" size="40"></li>
</ul>
<a href="javascript:void(0)" id="list3_add">項目を追加</a>

(CSS)
#list3_add {
	display:block;
	width:100px;
	padding:4px 8px;
	text-decoration:none;
	text-align:center;
	color:#fff;
	background:#24b;
	border:2px solid #37f;
	border-radius:4px;
}

(JavaScript)
$('#list3').addInputArea({
	cls_del : 'list3_del',
	btn_del : $('<img>').attr('src', 'img_del.png')
});

追加ボタン、削除ボタンはCSSや画像で装飾できます。

複数の入力欄を一括して増減



(HTML)
<ul id="list4">
	<li>
		<div>
			<label for="list4_name_0">名前</label>
			<input type="text" size="40" name="list4_name_0"><br>
			<label for="list4_mail_0">メールアドレス</label>
			<input type="text" size="40" name="list4_mail_0">
		</div>
	</li>
</ul>

(JavaScript)
$('#list4').addInputArea();

複数の入力欄を扱う場合も、特にオプションを必要としません。
なお、id,name属性だけでなく、for属性も増減に応じて末尾の番号が変化します。
